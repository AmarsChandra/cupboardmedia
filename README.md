# Cupboard Media

A modern website for Cupboard Media, a digital media company. Built with inspiration from Doing Things Media's design, featuring a brown color scheme and all the same sections.

## Project Structure

- `index.html` - Main HTML file with all sections
- `styles.css` - Stylesheet with brown color scheme and responsive design

## Sections Included

1. Header with logo and navigation
2. Hero section ("WE ARE CUPBOARD MEDIA")
3. Introduction and Statistics section
4. Brand Showcase section
5. WE CREATE section
6. WE GET RECOGNIZED section
7. WE HELP BRANDS section
8. Partner Logos section
9. WE SHOP TILL WE DROP section
10. FIND US IN RETAIL section
11. WE'RE FULLY LICENSED section
12. About Cupboard Media section
13. LET'S DO THIS contact section
14. Footer

## Color Scheme

- Primary Brown: #8B4513 (Saddle Brown)
- Dark Brown: #2c1810
- Medium Brown: #3d2818
- Light Brown/Beige: #d4a574
- Background: #fff, #f5f5f5

## Getting Started

Simply open `index.html` in a web browser to view the website.

## Customization

Update the content in `index.html` and customize the colors in `styles.css` to match your brand needs.
